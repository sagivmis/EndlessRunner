Animated Coins Pack
This pack contains high quality animated 2D coins assets with 60 coins.
These coins have 6 types, and every type coin has a 10-frame animation ready for use.  
Perfect when creating a new 2D Platformer or isometric style game.

Coins are provided in .png and .ai format. 
Illustrator Source files include all coins texture data.
And you can create new assets based on them!
